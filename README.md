# AI-sudoku-Solver-Comparative-Analysis-of-OCR-CNN-and-Hybrid-Approaches-for-Automated-Sudoku-Solver
Image recognition sudoku solver with 3 different models to see which is most efficient


# Link to the dataset used
https://drive.google.com/drive/folders/1vg_eTraqsKGyCOn0KRdcKMF8Y2BSyOKf?usp=sharing

The code is in the ipynb file which makes it only executable on google colab. So please use google colab to run the code.

Step 1:
Open the link above provided, download the dataset and upload it onto your google drive

Step 2:
Open the Final_Combined_solution.ipynb file on google colab

Step 3:
Make sure to change the "data_dir" variable's filepath to where you upload the dataset onto in your drive

Step 4:
Click "run all" to run the entire code for the first run

Step 5:
Make sure that all cells executed have the same output as provided in the .ipynb file which you can refer to by opening it in github

Step 6:
In the last cell, change the ground truth to that of the image that you want to test it with. If you want to test it with what we tested, you 
can download and use upload when running the last cell in the "test_image" file provided.

Step 7:
Enjoy the Automatic Sudoku Solver!!
